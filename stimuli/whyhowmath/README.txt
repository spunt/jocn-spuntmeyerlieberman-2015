Meaning of digits in stimulus file name:

Digits 1-3: stimulus ID (starts at 101)
Digit 4 = Statement Type {0 : 'Body-Focused', 1: 'Mind-Focused', 2: 'Math'}
Digit 5 = Statement-Image Appropriateness {0: 'Inappropriate', 1: 'Appropriate'}

For example, 11601.jpg is stimulus ID 116, included a body-focused statement that was appropriate to the image.
